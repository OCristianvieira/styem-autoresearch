## 🧠 FÓRMULAS DE CONVERSÃO COMPROVADAS

### Fórmula #1: NEGAÇÃO TRIPLA
```
"O melhor [X] para [resultado] NÃO é [popular 1], não é [popular 2], e definitivamente não é [popular 3]."
```
**Por que funciona:** Quebra padrão, cria curiosidade, posiciona como especialista contrarian.

### Fórmula #2: REVELAÇÃO PROIBIDA
```
"[Autoridade] odeiam que eu conte isso, mas [revelação]..."
```
**Por que funciona:** Cria "nós vs eles", sensação de informação privilegiada.

### Fórmula #3: INVESTIMENTO VALIDADO
```
"Gastei $[valor alto] para descobrir que [insight]..."
```
**Por que funciona:** Prova que você fez o trabalho pesado, reduz risco percebido.

### Fórmula #4: MÉTRICA CHOCANTE
```
"Foi comprovado que [solução] supera [alternativa] em [métrica] em [%]%."
```
**Por que funciona:** Números específicos constroem credibilidade, não são vagos.

### Fórmula #5: TRANSFORMAÇÃO TEMPORAL
```
"Eu recomendo há [tempo]. [Período curto] depois, sempre voltam [emoção forte] porque [resultado]."
```
**Por que funciona:** Mostra consistência (tempo longo) + resultado rápido (período curto).

### Fórmula #6: ESCASSEZ REAL
```
"Quando acaba, [consequência]. E acaba [frequência/velocidade]."
```
**Por que funciona:** Escassez real (não fake) dispara urgência genuína.

   - TikTok: 15-30s

   - Qual hook teve melhor retenção?
   - Qual plataforma converteu mais?
   - Qual persona gerou mais vendas?
   - Dobrar no que funciona

**Total: 90 min por roteiro completo**

   ```
   Versão C: Médica 35+ moderna, laboratório

   ```

   ```
   Versão C: 60-90s (Completo com história)
   ```

   **Métrica:** Retenção completa + Conversão

   ```
   TikTok vs Instagram vs Facebook
   Mesmo roteiro, mesma persona, mesmo produto
   ```
   **Métrica:** ROAS por plataforma
