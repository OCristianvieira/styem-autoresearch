## 🧠 FRAMEWORKS QUE SIGO

### Framework 1: Tipos de Headlines (para otimizar)

```
1. CURIOSIDADE + NÚMERO
   "3 coisas que ninguém te conta sobre [tema]"

2. PROMESSA + TIMEFRAME
   "Como [resultado] em [tempo] mesmo se [limitação]"

3. CONTRASTE/SURPRESA
   "Fiz [errado] e [resultado positivo]"

4. PROBLEMA → SOLUÇÃO
   "Você está [problema] (e a solução é [simples])"

5. FOMO/URGÊNCIA
   "Antes que [X] mude, [ação]"
```

### Framework 2: Estrutura dos 4 Blocos

```
BLOCO 1 - HOOK (3-5 seg)
├─ Pergunta que intriga
├─ Promessa do que vai aprender
└─ Transição para tema

BLOCO 2 - DESENVOLVIMENTO
├─ Seção 1: [tema principal]
├─ Seção 2: [aprofundamento]
└─ Seção 3: [aplicação]

BLOCO 3 - CTA MID-ROLL
├─ Menção natural de consultoria
├─ Oferta de diagnóstico
└─ Urgência leve (slots limitados)

BLOCO 4 - FINALIZAÇÃO
├─ Recap dos 3 pontos principais
├─ CTA FINAL (stronger)
└─ Encerramento com frase signature
```

### Framework 3: 3 Tipos de CTA de Consultoria

```
CTA SOFT (início/meio)
├─ Objetivo: Despertar interesse
├─ Tom: Conversacional, natural
└─ Exemplo: "Isso é o que trabalho com clientes..."

CTA MEDIUM (meio/fim)
├─ Objetivo: Oferecer valor (diagnóstico grátis)
├─ Tom: Educativo, oferta clara
└─ Exemplo: "Se quer um diagnóstico, manda DM..."

CTA STRONG (final)
├─ Objetivo: Converter em ação
├─ Tom: Urgência, exclusividade
└─ Exemplo: "Tenho 3 vagas esse mês, aproveita..."
```
